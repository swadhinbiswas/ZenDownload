# ZenDownload 0.1.0

Initial public release.

## Highlights

- **Multi-protocol download engine** — HTTP, HTTPS, BitTorrent, magnet links, HLS, DASH
- **Streaming from 1,462+ sites** — YouTube, Twitter/X, Reddit, TikTok, Instagram, Twitch, Vimeo, SoundCloud
- **Music downloader** — search YouTube & SoundCloud with full ID3 metadata
- **Speed test** — multi-server benchmark with live graph
- **Smart queue** — concurrency control, per-file speed limits, pause/resume
- **BitTorrent** — DHT, PEX, magnet resolution, sequential streaming playback
- **Debrid** — Real-Debrid, AllDebrid, Premiumize support
- **Plugin Store** — 40+ community plugins (Discord, Spotify, qBittorrent sync, IPTV browser, …)
- **Native dark UI** — 12 accent colors, glass effects, adjustable corner radius
- **Clipboard monitor** — auto-detect URLs with category preview
- **Scheduler** — time-based download windows
- **Watch folders** — auto-add dropped files
- **Subscriptions** — RSS, RSSHub, YouTube channels

## What's next

- iOS build (Tauri 2)
